# Project Overview

The **Blinkit Sales Analytics Dashboard** is a comprehensive Business Intelligence solution designed to analyze retail sales performance, customer preferences, outlet performance, and inventory distribution.

The dashboard transforms raw sales data into meaningful insights through interactive visualizations, KPIs, and dynamic filters, enabling data-driven business decisions.

It provides a complete overview of:

- Total Revenue
- Average Sales
- Product Performance
- Outlet Performance
- Customer Ratings
- Item Distribution
- Location Analysis
- Outlet Size Analysis

---

# Business Objective

This dashboard helps stakeholders answer critical business questions such as:

✔ Which outlet generates the highest revenue?

✔ Which item category performs best?

✔ How does outlet size affect sales?

✔ Which outlet type contributes the most sales?

✔ What is the average customer rating?

✔ Which locations have the strongest business performance?
